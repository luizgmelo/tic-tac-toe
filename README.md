## What is the project?
Tic Tac Toe is a game in which two players seek in alternate turns to complete a row, a column, or a diagonal with either three O's or three X's drawn in the spaces of a grid of nine squares </a>

## For what this project exists?
- Enjoy sometimes with a friend.
- Helps to learn python. 


## How to access the project?
```
git clone https://github.com/luizgmelo/tic-tac-toe.git
cd tic-tac-toe
python main.py
```
